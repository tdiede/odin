# odin

A Clojure library designed to ... well, that part is up to you.

## Usage

To start local development.

```
$ lein install
$ lein cooper
```

Start a REPL. In the `user` namespace invoke `go!`.

```
user> (go!)
```

Note the web server starts up on port 8082.
